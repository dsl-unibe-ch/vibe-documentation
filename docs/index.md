The VIBE desktop is the virtual platform for advanced and interactive image analysis for researchers of the University of Bern. It is powered by the HPC of the University of Bern (UBELIX) and it uses the OpenOnDemand framework for applications development.


The VIBE desktop uses [https://ondemand.hpc.unibe.ch](https://ondemand.hpc.unibe.ch).

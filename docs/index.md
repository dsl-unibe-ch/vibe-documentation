# Welcome to the **V**irtual **I**mage analysis of the University of **BE**rn (**VIBE**) User Documentation


The VIBE desktop is the virtual platform for advanced image analysis powered by the high performance computer (HPC) cluster of the University of Bern. 


The VIBE desktop uses [https://ondemand.hpc.unibe.ch](https://ondemand.hpc.unibe.ch).
